# Frontend React Test

Build with Bun. Supports Docker in case of any issues with Bun or other package managers.

## Live preview

<https://frontend-react-test-theta.vercel.app/>

## Prerequisites

- [Bun](https://bun.sh/) (v1.x) — or [Docker](https://www.docker.com/).

## Getting started

```bash
# bun/npm/pnpm/yarn...
bun install
bun run start

# or:
docker compose up
```

## Running tests

```bash
bun run test

# or:
docker compose run --rm test
```
